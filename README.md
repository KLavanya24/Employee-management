# Employee Management System (EMS)

## 📌 Project Description
This is a simple Employee Management System developed using HTML, CSS, and JavaScript.  
It allows users to manage employee details like ID, Name, Department ID, Salary, and Hire Date.

---

## 🚀 Features
- 🔐 Login Authentication (Username & Password)
- ➕ Add Employee
- ✏️ Edit Employee
- ❌ Delete Employee
- 🔍 Search Employee
- 📊 Salary Graph Visualization
- 🌙 Dark Mode Toggle

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js (for graph)
